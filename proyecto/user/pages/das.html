Para poner una letra desplazada se pone i. Se usa bpara poner en negrita. U sirve para subrayar.
Salto de linea con br.
Linea horizontal con hr. Mark sirve para resaltar texto. Small sirve para texto pequeño. Del sirve para texto tachado. Ins sirve para texto insertado. Sub sirve para texto subindice. Sup sirve para texto superindice.
